Market Simulator & Computational Investment

I built a stock market simulator which takes a starting capital along with buy and sell orders for dates in the past and then evaluate how much money was made by the given portfolio over a givne time period using data acquired from Yahoo Finance API.

After making the simulator I conducted event studies where I generated buy and sell orders using Bollinger Bands which basically predict when stock prices go above and below their typical value. Based on this I generated buy and sell orders whose performance I measured using the above simulator. I also used some machine learning algorithms to cluster which was the best time periods and best stocks suited to such trades.

  • Role: Developer

TransitCheck

TransitCheck is a web application I made along with a team for BattleHack 2014, a hackathon organized by PayPal. Quite often we arrive on time for a bus but miss it because it has arrived a minute too early. These situational delays in the TTC can often impact a lot of people adversely. TransitCheck is an application to help with this need.

Whenever a user detects an anomaly on a given network he/she informs all others who use that bus route of the small change. In return other users can give the first person money as thank you using the PayPal API.

I mainly wrote a web crawler to acquire data off the TTC website using some Java Libraries. I also helped in the frontend of the web application.

BrailleStorm

We won the National Robot Olympiad (Dubai) by building BrailleStorm, a fully functional, semi-autonomous robot which uses a simplistic yet sophisticated design similar to an industry-standard printer. Using a three-axis movement mechanism, a programmable pin cartridge imprints Braille characters in real-time using a Bluetooth user-input NXT device.

Primarily made using LEGO Mindstorms NXT Education kit components, the robot is an innovative and cost-effective solution. Besides being portable and easy-to-assemble, the imprint accuracy and efficient printing make it a cost-effective communication alternative.Implementing real-world design concepts, it consists of over 60 gears in various advanced gear-train mechanisms to enable triple-axis movements. The model also exhibits structural symmetry in all the components, such as the dual-pin pricking cartridge, linear axis movement beams and paper feeder. Most of the code used to implement this project was written using LeJOS, a Java based language for progrmaming LEGO robots.

  • Date: Nov 2012
  • Role: Project Manager & Developer
